Permutation
===========

Features
--------

This module provides permutation variants of the following statistical tests:

  * t test
  * Wilcoxon rank sum test
  * One-way ANOVA F test
  * Kruskal-Wallis test
  * Kolmogorov-Smirnov test
  * Correlation test
